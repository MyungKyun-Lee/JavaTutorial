수업시간에 했던 소스 또는 장바구니 과제 소스 중 하나 선택하여 작성

1. 제목   :   장바구니 상품 생성/삭제/합계 기능구현
               (제공된 화면에서 상품 장바구니 기능을 자바스크립트로 구현)
2. 구현한 기능 
   (1) 주문하기 - 상품정보 입력값 체크 및 장바구니 addRow() 기능
   (2) 장바구니 계산표시 - 상품금액 계산 및 표시 formatCurrency(num) 기능
   (3) 선택상품삭제 - 다중 선택된 상품의 삭제 delRow() 기능
   (4) 전체상품 선택 - 전체상품 선택 체크박스의 값에 따라 표시 allChk1() 기능

3. 구현과정 풀이
   (1) 주문하기 : addRow()
       - 상품정보 및 가격, 배송비 입력값 체크하여 없는 경우 메시지 팝업후 중단
       - 장바구니 Table의 insertRow() 추가, 추가된 Row에 각<td>태그 insertCell() 추가
         각 cell에 맞는 html 및 입력값 표시
       - [선택상품금액] 계산(수량*상품금액)된 금액 표시
         [배송비] (배송비) 금액 표시
         [주문금액] 계산(선택상품금액 + 배송비)된 금액 표시
       - 입력값 다음 사용을 위해 초기화
       - 장바구니 입력세시지 숨기
   (2) 장바구니 계산표시
       - 금액 표시 : 숫자는 천단위 ',' 추가 및 '원' 문자 추가 formatCurrency(num) 기능
   (3) 장바구니 선택상품삭제 : delRow() 
       - 장바구니 row 수 체크 없음면 메시지 출력
       - 선택 체크된 상품 조건으로 삭제처리
       - 삭제처리 전 [선택상품금액],[배송비],[주문금액] 금액 수정
       - 장바구니 모두 빈 상태 이면 장바구니 메시지 보이기
   (4) 전체상품 선택
       - 전체상품 선택이 값을 모든 장바구니 선택값에 입력 (true이면-> 모두true)
       - 테이블의 row.length 수 만큼 처리 (for문)
